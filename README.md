Example ABC News application in react-native
========================

### Requirements, install as needed:

##### React Native:
```
$ npm i -g react-native-cli
```
##### clone this repo:

```
$ cd react-native-abc-news
$ npm install
$ react-native run-android
```

Open the directory in the text editor of your choice.
